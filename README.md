# NodePhone
A minimum command line voice chat app by node


# server

install:

```
npm i ws
```

run:

```
node sv.js
```


# app

macos install:

Tested on node v16.13.0 and macos 12.3.1, using homebrew

```
arch -x86_64 zsh
npm i node-record-lpcm16
npm i speaker  # this need x86_64
```

run:

```
node phone.js
```
